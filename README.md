# Holyheld SDK

Holyheld SDK provides methods to offramp and onramp crypto to and from Holyheld account in Typescript/Javascript environment. By using provided functions, the process is split in only a few steps.

## Installation

Using npm:

```bash
$ npm install @holyheld/sdk
```

Using yarn:

```bash
$ yarn add @holyheld/sdk
```
